Ensinamentos de como percorrer e  criar pastas pelo prompt de comando 

Windows
cd - navega entre as pastas
dir - lista diretórios
mkdir cria novas pastas
del - deleta arquivos 
rmdir - deleta pastas

Unix
cd - navega entre as pastas
ls - lista diretórios
mkdir cria novas pastas
rm - deleta arquivos 
rf - deleta pastas
